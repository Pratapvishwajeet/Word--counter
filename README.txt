Word Counter

This repository contains a Python-based Word Counter.Features of the Programs are:

Punctuation Handling:
The program removes punctuation from the input text before counting words, ensuring accurate word counting.



Statistics Calculation:
The program calculates additional statistics, including the number of unique words and the average word length.


Multiple Inputs:
Users can analyze multiple texts in a single session. After analyzing each text, users can choose to input another text or finish.


Final Summary:
At the end of the session, the program provides a summary of the word count, unique words, and average word length for all the combined texts entered during the session.
